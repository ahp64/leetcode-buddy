👋 Set up your streak

Everything — members, timezone, reminders — is set with one secret on your repo:

  1. Open Settings → Secrets and variables → Actions → Repository secrets tab → add a secret named BUDDY_CONFIG: 
    {
  "settings": { "timezone": "America/Chicago", "reminderHours": [12, 18, 21] },
  "members": [
    { "name": "You", "leetcodeUsername": "your-leetcode-username",
      "email": "you@example.com", "notifyEmail": true },
    { "name": "Buddy", "leetcodeUsername": "their-leetcode-username",
      "email": "buddy@example.com", "notifyEmail": true }
  ]
}
  2. Add reminders too, so people actually get pinged — same place, secrets named SMTP_HOST/SMTP_PORT/SMTP_USER/SMTP_PASS/SMTP_FROM for email and/or TWILIO_ACCOUNT_SID/TWILIO_AUTH_TOKEN/TWILIO_FROM for text. Details in the README.
  3. Run the dashboard workflow once to pick it up: Actions → Streak check & dashboard → Run workflow (it also runs itself automatically every 15 minutes).
